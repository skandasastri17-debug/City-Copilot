# City Copilot

City Copilot is a Toronto-focused AI assistant that helps residents report city issues, find local resources, and suggest neighborhood improvements. It turns plain-language requests into clear answers, department routing, draft reports, next steps, and live data/resource leads.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js.

## Core routes

- `/` chatbot-first City Copilot experience
- `/assistant` same assistant workspace
- `/reports` mock report tracking
- `/resources` city resource finder
- `/about` hackathon/demo explanation
